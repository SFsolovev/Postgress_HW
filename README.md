# Postgress_HW
[Сессионное задание] Итоговый проект. База данных
Задание выполнялось на Postgresql 18, редактор DBeaver 26.

Содержит 4 папки:
1. Транспортные средства (1.Vehicles)
2. Автомобильные гонки (2. Car racing)
3. Бронирование отелей (3. Hotel booking)
4. Структура организации (4. Organization structure)

В каждой папке три скрипта:
1 - Создание таблиц.
2 - Заполнение таблиц.
3 - SQL запросы с решением.
